# Code of Conduct

This project follows the Contributor Covenant.

## Our Standards

- Be respectful in all interactions.
- Give actionable, technical feedback.
- Assume good intent and focus on code quality.

## Unacceptable Behavior

- Harassment, insults, discrimination.
- Personal attacks or trolling.
- Publishing private information.

## Enforcement

Report incidents to project maintainers via private issue/email.

Maintainers may remove comments, close threads, or block contributors for serious violations.
