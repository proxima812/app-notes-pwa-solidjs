# Модуль `components`

## Ответственность

`components` содержит UI-блоки с минимальной бизнес-логикой:
- `NotesHeader` — верхняя панель управления
- `EditorPanel` — создание/редактирование заметки
- `NoteCard` — карточка заметки
- `BackupPanel` — инструменты импорта/экспорта
- `EmptyState` — пустое состояние
- `icons` — локальные SVG-иконки

## Границы гидрации

Проект работает как SPA на Solid, поэтому компоненты должны оставаться легковесными и переиспользуемыми.

## Ограничения производительности

- Не добавлять тяжелые вычисления в JSX.
- Переиспользовать утилиты из `lib` вместо дублирования логики.

## Подводные камни

- Для адаптивности обязательно проверять мобильные ширины (360px+).
- Кнопки действий должны иметь стабильный контраст в light/dark режимах.
